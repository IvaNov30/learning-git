## Разбор большой функции decode_bytes

Что делает: Декодирует байты в строку, определяет тип переноса строк и кодировку

Основные шаги:
1. Определяет кодировку через tokenize.detect_encoding
2. ЕСли передана encoding_overwrite - использует её
3. Определяет тип переноса строк (CR/LF/CRLF) по первой строке
4. Декодирует весь файл с обратной кодировкой
